# ポートフォリオ `/`

## 概要

認証不要の公開ポートフォリオページ。個人情報・スキル・経歴を掲載する静的ページ。

## コンポーネント構成

```
App.tsx (portfolio)
├── Header.tsx
│   ├── ナビゲーションリンク
│   └── ソーシャルリンク (GitHub / X / LINE)
├── Section: Profile
│   ├── プロフィール写真
│   ├── 名前 (佐藤 康樹)
│   └── 自己紹介
├── Section: Education
│   ├── N高等学校 (2024/3卒)
│   └── 武蔵野大学 (2024/4〜)
├── Section: Career
│   └── HALLOプログラミングスクール (2024/6〜)
└── Section: Skills (placeholder)
```

## 状態管理

なし（完全静的）

## 遷移フロー

```
外部アクセス
  → / (ポートフォリオ)
  ヘッダーナビ → セクションスクロール
  ※ /app へのリンクなし（独立したサイトとして機能）
```
